# AStockAgents 功能规划方案

> 以机构投资经理视角规划，覆盖"选股→研究→决策→交易→风控→复盘"全链路

---

## 一、产品定位

### 1.1 目标用户

| 用户角色 | 核心诉求 |
|---------|---------|
| **基金经理** | 快速筛选标的、深度研究、辅助决策、组合管理 |
| **研究员** | 高效覆盖、多维分析、研报生成、跟踪预警 |
| **交易员** | 执行信号、风控检查、交易记录 |
| **风控经理** | 实时监控、风险预警、合规检查 |
| **个人投资者** | 专业级分析工具、降低信息不对称 |

### 1.2 核心价值主张

```
传统投研流程:  选股 → 研报 → 讨论 → 决策 → 执行 → 复盘
                2天    1周    半天   1天    10分钟  1周

AStockAgents:   选股 → AI多维分析 → 多空辩论 → 辅助决策 → 信号推送 → 自动复盘
                5分钟  30秒       1分钟    实时      实时     自动
```

## 三、功能模块详细设计

### 模块一：🔍 智能发现

#### 1.1 智能选股

| 功能 | 描述 | 用户操作 | 输出 |
|------|------|---------|------|
| **条件选股** | 多维度筛选（PE/PB/市值/涨幅/换手率/行业） | 设置筛选条件 → 点击筛选 | 符合条件的股票列表 |
| **技术形态选股** | 基于K线形态筛选（金叉、突破、底部反转等） | 选择形态类型 → 执行 | 形态匹配的股票 |
| **资金面选股** | 主力净流入/北向资金/融资余额变化 | 设置资金条件 → 执行 | 资金异动股票 |
| **AI智能选股** | 多因子综合评分，AI推荐 | 点击"AI推荐" → 设置偏好 | 推荐股票+推荐理由 |
| **策略选股** | 保存/加载选股策略模板 | 保存当前条件为模板 | 可复用的策略模板 |

#### 2.2 技术分析专项

| 功能 | 描述 |

## 四、用户旅程设计

### 4.1 首次使用引导

```
步骤1: 注册/登录
  ↓
步骤2: 选择投资风格 (保守/稳健/积极/激进)
  ↓
步骤3: 添加自选股 (搜索/推荐/导入)
  ↓
步骤4: 体验一键分析 (选一只股票体验全维分析)
  ↓
步骤5: 设置预警 (对自选股设置价格预警)
  ↓
步骤6: 配置通知 (选择通知方式)
```

### 4.2 日常使用流程

```
早上 8:30 ──→ 查看隔夜新闻/外盘/公告
  ↓
早上 9:00 ──→ 查看自选股预警/异动提醒
  ↓
盘中 9:30 ──→ 实时监控持仓/接收信号
  ↓
盘中 10:00 ──→ 发现机会 → 一键分析 → 决策
  ↓
盘中 14:00 ──→ 查看板块轮动/资金流向
  ↓
收盘 15:00 ──→ 查看当日总结/AI复盘
  ↓
晚上 20:00 ──→ 深度研究/策略优化/回测
```

### 4.3 专业用户进阶流程

```
第1周: 熟悉基础功能 → 建立自选股池
  ↓
第2周: 使用策略回测 → 验证交易策略
  ↓
第3周: 配置预警系统 → 自动化监控
  ↓
第4周: 自定义智能体 → 个性化分析
  ↓
第5周: 组合管理 → 多股票协同分析
  ↓
持续: 知识积累 → 策略迭代 → 收益提升
```
